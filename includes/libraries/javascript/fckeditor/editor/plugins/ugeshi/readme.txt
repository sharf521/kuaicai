使用方法：

1. 将ugeshi文件夹存放于fckeditor/editor/plugins/目录下；

2. 编辑fckconfig.js文件，添加 FCKConfig.Plugins.Add('ugeshi', 'zh-cn');

3. 设置FCKEditor工具栏，FCKConfig.ToolbarSets["Default"]值中添加：UGeSHi (注意大小写)

清空浏览器临时文件，开始体验吧...
